Welcome to Match The Graph, your goal is to select the set of disjoint edges (two edges cant share the same vertex) that maximizes the sum of weights

Intuition

  • Set A (red) is job applicants, Set B (blue) is companies. The edge weights (between [-1, 1]) represent how suitable a person is for the job they applied to
  • Red and and blue are two groups of people. The weight on each edge represents the compatibility score between each person of the other group
  • Set A is people calling an Uber, and B is Uber drivers. The edge weight represents how convenient it is for the driver to pick up the passenger

Instructions

  • Click and drag nodes to move them around
  • Click on edges weights (the numbers) to select/unselect them in the matching
  • Use the Check button to verify if your matching is optimal
  • Use Reset to start over with new random weights
  • Change the number of points in each set on the top right
